# Offline basemap extracts

`DE.pmtiles`, `FR.pmtiles`, and `IE.pmtiles` are bounded zoom 0–8 extracts
from the Protomaps daily OpenStreetMap basemap build dated 2026-07-18.

- Source: https://build.protomaps.com/20260718.pmtiles
- Data: © OpenStreetMap contributors
- Database license: Open Database License (ODbL)
- Extract tooling: Protomaps PMTiles CLI

The application displays the OpenStreetMap and Protomaps attribution whenever
these tiles are rendered.
