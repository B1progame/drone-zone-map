# Offline architecture

The service worker caches the versioned application shell and same-origin generated data. Offline country and regional packages use IndexedDB chunk and tile stores, not localStorage. Packages are generated by country, region, rectangle, or radius and can include zone layers and permitted basemap tiles.

Downloads are generation-based: new chunks are written first, the package pointer changes only after completion, and prior chunks are then removed. Interrupted or quota-failed downloads delete the incomplete generation and preserve the previous valid package.

Every package carries authority, timestamps, layer selection, warnings, and the reminder that live temporary restrictions and weather may be absent.
