# Translation system

The UI supports the requested English, German, French, Spanish, Italian, Portuguese, Dutch, Norwegian, Swedish, Danish, Finnish, Polish, and Czech languages (plus existing additional languages).

`public/data/translations/aviation-terms.json` is a generated reviewed terminology dictionary and stable-hash offline cache. Official names and text are retained. Zone IDs, legal articles, callsigns, URLs, emails, and technical identifiers are never translated. Missing text falls back to the original official language.
